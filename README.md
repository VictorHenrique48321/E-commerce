<h1>Como instalar</h1>

<h5>Utilize o comando npm install nas pastas Client e Server</h5>

<h5>Inice o servidor dentro da pasta server com :</h5>

```sh
node server.js
```
<h5>Depois dentro da pasta client inicie o site com :</h5>

```sh
npm start
```
